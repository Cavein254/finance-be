About

The whole concept of this application is to provide developers with enough historical data to test their ideas for FREE. Most other sites have a restriction on the number of API calls one can make hence limiting.

Queries

getAllSymbols

Response

Returns a StockSymbolsResults

Example

Query
query GetAllSymbols {
  getAllSymbols {
    success
    data {
      id
      name
      ticker
    }
    error
  }
}
Response
{
  "data": {
    "getAllSymbols": {
      "success": true,
      "data": [StockTicker],
      "error": "abc123"
    }
  }
}

Queries

getHistoricalData

Response

Returns a GetStockDataResponse

Arguments
Name Description
symbol - String!

Example

Query
query GetHistoricalData($symbol: String!) {
  getHistoricalData(symbol: $symbol) {
    success
    data {
      id
      stockId
      date
      open
      high
      low
      close
      openInt
    }
    error
  }
}
Variables
{"symbol": "xyz789"}
Response
{
  "data": {
    "getHistoricalData": {
      "success": false,
      "data": [StockData],
      "error": "xyz789"
    }
  }
}
